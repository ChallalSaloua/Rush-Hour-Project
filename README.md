### 🚗 Rush Hour Solver – Artificial Intelligence Problem-Solving Project

Rush Hour Solver is an Artificial Intelligence project designed to solve the classic Rush Hour puzzle using state-space search and heuristic-based problem-solving techniques.

The objective of the game is to move vehicles within a congested grid in order to clear a path for the target vehicle and allow it to exit the board. The project models the puzzle as a search problem, where each board configuration represents a state and each vehicle movement represents a valid action leading to a new state.

The system explores possible configurations, evaluates alternative solutions, and applies intelligent search strategies to find efficient paths toward the goal state. Different heuristic functions are used to estimate solution quality and guide the search process while reducing computational complexity.

#### Key Features

* Interactive Rush Hour puzzle solver
* State-space representation and exploration
* Automatic solution generation
* Heuristic-based decision making
* Search tree visualization and analysis
* Performance evaluation of search strategies
* Multiple puzzle configurations and difficulty levels

#### Educational Objectives

This project was developed as part of an Artificial Intelligence and Problem Solving course to study how intelligent agents can solve complex constraint-based puzzles. It demonstrates the practical application of search algorithms, heuristic evaluation functions, and state-space exploration techniques for finding optimal or near-optimal solutions.

#### Learning Outcomes

* Modeling real-world problems as search spaces
* Designing effective heuristic functions
* Evaluating search efficiency and solution quality
* Understanding the trade-off between search cost and optimality
* Applying AI techniques to combinatorial optimization problems
